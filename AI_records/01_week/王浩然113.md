## 周三 — 前端开发

**王浩然**：axios 怎么封装比较好，我想统一处理 token 和报错

**AI**：用拦截器，请求时自动带 token，响应时统一判断 code

**王浩然**：帮我写个大概的

**AI**：

```typescript
const instance = axios.create({
  baseURL: '/api',
  timeout: 30000,
});

instance.interceptors.request.use(config => {
  const token = localStorage.getItem('token');
  if (token) config.headers.Authorization = `Bearer ${token}`;
  return config;
});

instance.interceptors.response.use(
  res => {
    if (res.data.code !== 0) {
      Toast.show({ content: res.data.message, icon: 'fail' });
      return Promise.reject(res.data.message);
    }
    return res.data;
  },
  err => {
    Toast.show({ content: '网络异常', icon: 'fail' });
    return Promise.reject(err);
  }
);
```

**王浩然**：那登录之后 token 存哪里

**AI**：localStorage，存 'diet_token' 这个 key

**王浩然**：采购清单要按类别分组，但后端返的是平铺数组，前端怎么分组

```typescript
const grouped: Record<string, ShoppingItem[]> = {};
shoppingList.forEach(item => {
```

**王浩然**：Vite 开发时端口 3000，怎么代理后端 8080

**AI**：vite.config.ts 里配 proxy

```typescript
server: {
  port: 3000,
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true,
    }
  }
}
```

前端请求 `/api/xxx` 自动转到 `localhost:8080`
