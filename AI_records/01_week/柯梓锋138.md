## 周四 — 后端 + 部署

**柯梓锋**：配餐算法中菜品池的设计，帮我看看有没有遗漏

**AI**：每道菜需要几个字段：
- name（菜名）
- category（分类：staple/meat/seafood/veg/soup）
- conflicts（和哪些忌口冲突）
- ingredients（食材 + 用量 + 类别）

然后筛选时：分类匹配 + 忌口不冲突 → 随机取 N 道

**柯梓锋**：如果筛出来不够怎么办

**AI**：加兜底，不限制分类只靠忌口过滤

```python
if not candidates:
    candidates = [d for d in pool
        if not any(r in d.conflicts for r in restrictions)]
```
