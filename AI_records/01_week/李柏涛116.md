**李柏涛**：采购清单怎么生成

**AI**：遍历三天三餐，只收集 `isExisting=False` 那道菜的食材，用 dict 去重

```python
shopping_map = {}
for day in days:
    for meal in day.breakfast + day.lunch + day.dinner:
        if not meal.isExisting:
            for ing in meal.ingredients:
                shopping_map.setdefault(ing.name, ing)
shopping_list = list(shopping_map.values())
```

**李柏涛**：setdefault 什么意思

**AI**：如果 key 不存在就设值并返回，存在就返回已有值。效果就是同名食材只加一次

@app.post("/api/recipes/upload")
async def upload_recipe(file: UploadFile):
    content = await file.read()

**李柏涛**：删除菜谱怎么判断 id 不存在

**AI**：

```python
@app.delete("/api/recipes/{recipe_id}")
def delete_recipe(recipe_id: str):
    before = len(recipes)
    recipes[:] = [r for r in recipes if r.id != recipe_id]
    if len(recipes) == before:
        raise HTTPException(status_code=404, detail="菜谱不存在")
    return ok(None, "已删除")
```

**李柏涛**：`recipes[:] = ` 和 `recipes = ` 有什么区别

**AI**：`recipes[:] = ` 原地修改列表，不改变引用。如果别的变量也指向这个列表，能看到更新。`recipes = ` 是新建列表
